Overview

Your code must be in the following syntax in order to work: 
function yourScript() {{ '{' }}
	
	// your logic goes here...

	return 'guess';  // string value of what your guess is
{{ '}' }}

Keep all code in a single function (inner functions are allowed).

If your code returns an invalid guess (like a code that doesn't exist, or that has already been guessed) it will end the game.

You have access to the this variable to save state between guesses if you wish, as well as access to game state.

Logging is available through console.log(message). You can also use console.error(message), console.warn(message), and console.debug(message).

All built-in game state variables are backed up prior to, and restored after, your function runs. This prevents tampering.

"Likeness"

A word's likeness is determined by whether or not the string's individual character values of the word match with the answer word's values in both index of the string, and in character value.

e.g. If "ghost" is the answer, and you guess "ghast", the likeness would be 4. However, if you guessed "roast", the likeness would only be 2, as only "s" and "t" would match in both index and value.

Functions

Function
Description
self.getCodes() The list of possible codes you can guess. The array of codes returned from this function are not string values, they are objects with three fields on them, code, isGuessed, and likeness. The code field on each object tells you the actual string value, the isGuessed field tells you whether or not you have guessed that code yet, and finally the likeness field tells you the likeness to the answer (a -1 means it hasn't been guessed yet, so there isn't a likeness assigned).
